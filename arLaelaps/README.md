# arLaelaps
AutoRally Robot Rover - Laelasp Test Platform

## Hardware
* Board: Odroid XUv
* CPU: 4 Core Arm7 + 4 Core Arm15 + GPU
* Motors: 4 brush DC motors
* Motor Controller: 2 RoboClaw 
* Naze32 IMU
* 3 ToF VL6180 sensors

## System
* OS: Ubuntu 14.04
* ROS: indigo
* RNR Laelaps

## Network
* hostname: laedev
* IP: 192.168.1.26
* ROS master
