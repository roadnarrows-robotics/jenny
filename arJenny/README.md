# arJenny
AutoRally Robot Rover - PRS Grocery Cart

## Hardware
* Board: Odroid XUv
* CPU: 4 Core Arm7 + 4 Core Arm15 + GPU
* Motors: 2 brush DC motors
* Motor Controller: RS160D
* Naze32 IMU
* 5 Ultrsonic sensors

## System
* OS: Ubuntu 14.04
* ROS: indigo
* RNR Kuon hybrid

## Network
* hostname: jenny
* IP: 192.168.1.26
* ROS master
